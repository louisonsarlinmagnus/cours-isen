## Résumé
| Effet        | Md           | HTML  |
| ------------- |:-------------:| -----:|
| Titre      | `#` (jusqu'a 6) | `<h1>` (jusqu'a `<h6>`) |
| **Gras**      | `**text**`      | `<b>text<b>` |
| <ins>Souligner</ins> |   ....    |  `<ins>text</ins>` |
| *Italique* | `*text*` | `<i>text</i>` |
