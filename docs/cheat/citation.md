## Blocs de citation

!!! example "Bloc de citation"

    === "Markdown"
        ```md
        > Un problème sans solution est un problème mal posé.  
        > Albert Einstein
        ```
    === "HTML"
        ```html
        <blockquote>
        Un problème sans solution est un problème mal posé.<br>
        Albert Einstein
        </blockquote>
        ```
    === "Rendu"
        > Un problème sans solution est un problème mal posé.  
        > Albert Einstein
